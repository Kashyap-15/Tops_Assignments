HTML elements are the building blocks of HTML pages. with HTML constructs,images and other objects such as interactive forms may be embedded into the rendered page. HTML provides a means to creat structured documents by denoting structural semantics for text such as headings, paragraph, lists,quotes and other items. HTML elementsnare delineated by tags, written using angle brackets. Tags such as and directly introduce content into pages. other tags such as surround and provide information about document text and may include other tags as sub-elements. Browsers do not display the HTML tags but use them to interpret the content of the page.
